#### Setup

<details>
<summary>G++</summary>

Install the G++ compiler:

`apt install build-essential g++`
<details>

#### Run

<details>
<summary>CPU</summary>

Compile the CPU example:

`g++ cpu.cpp library/functions.cpp -o cpu.bin -std=c++17 && ./cpu.bin`
